# Data-Professional-Survey-Analysis
This project presents an interactive Power BI dashboard analyzing survey data from data professionals across different countries, roles, and experience levels. The goal is to uncover insights into salaries, demographics, job satisfaction, and industry trends.

Key Insights are;
1.Total Survey Respondents and demographic distribution
2.Average Age of Respondents
3.Average Salary by Job Title and Gender
4.Job Satisfaction Analysis
5.Happiness with Salary
6.Work-Life Balance
7.Geographical Distribution of Respondents
8.Favorite Programming Languages

Tools & Technologies
1.Power BI (Data visualization & dashboard creation)
2.Power Query (Data cleaning & transformation)
3.DAX (Calculated measures)

The dataset contains survey responses from data professionals, including information on salary, job roles, programming preferences, and satisfaction levels.

Project Objective;
To analyze trends in the data profession and provide insights that can help aspiring and current data professionals understand the industry landscape.
