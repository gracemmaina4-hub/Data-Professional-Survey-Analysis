# 📊 Survey Analysis: Data Professionals

## 📌 Project Overview

This project presents an interactive data analysis dashboard built using **Power BI**, focused on understanding trends among data professionals. The analysis explores key aspects such as salary distribution, demographics, programming preferences, and job satisfaction.

The goal of this project is to derive meaningful insights from survey data and communicate them effectively through clear and visually appealing dashboards.

---

## 🎯 Objectives

* Analyze salary trends across different data-related job roles
* Understand demographic distribution (age, gender, country)
* Identify popular programming languages among professionals
* Evaluate satisfaction levels with salary and work-life balance
* Showcase data storytelling and visualization skills

---

## 📂 Dataset Description

The dataset contains survey responses from **630 data professionals**, including:

* 🌍 Country of respondents
* 🎓 Job titles (Data Analyst, Data Scientist, Data Engineer, etc.)
* 💰 Salary information
* 👩‍💻 Favorite programming languages
* 👥 Gender distribution
* 🎂 Age of respondents
* 😊 Satisfaction metrics (salary & work-life balance)

---

## 📊 Key Insights

### 💰 Salary Analysis

* **Data Scientists** have the highest average salary (~90)
* Followed by **Data Architects** and **Data Engineers**
* **Data Analysts** earn comparatively lower average salaries

### 🌍 Geographic Distribution

* Majority of respondents come from:

  * United States
  * India
  * United Kingdom
  * Canada

### 👩‍💻 Programming Languages

* **Python** is overwhelmingly the most preferred language
* Other languages like **R**, **C/C++**, and **JavaScript** trail behind significantly

### 👥 Gender Distribution

* Nearly equal representation:

  * Male: ~51%
  * Female: ~49%

### 🎂 Age Insights

* Average age of respondents: **~29.87 years**

### 😊 Satisfaction Levels

* Work-life balance satisfaction: **5.74 / 10**
* Salary satisfaction: **4.27 / 10**
* Insight: Professionals are moderately satisfied with work-life balance but less satisfied with salary

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Interact with filters and visuals to explore insights

---

## 💡 Key Takeaways

* Data Science roles remain the highest paying in the industry
* Python dominates as the primary programming language
* There is a noticeable gap between salary expectations and satisfaction
* The data industry is relatively young with an average age below 30
